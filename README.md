# Ex 01

First example of [Programming and Development Paradigms - a.y. 2025-2026](https://www.unibo.it/en/teaching/course-unit-catalogue/course-unit/2025/526526) ([Computer Science and Engineering](https://corsi.unibo.it/2cycle/ComputerScienceEngineering)).

## Details

Refactoring a legacy Bank Account codebase using Test‑Driven Development (TDD) with JUnit.  
The goal is to apply the Template Method Pattern to design and test two account types: a **Silver** Bank Account and a **Gold** Bank Account.
